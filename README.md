# React Tabs

## Installation

Install `@spkf/react-tabs` using your preferred package manager:

### npm

```bash
npm install @spkf/react-tabs
```

### pnpm

```bash
pnpm add @spkf/react-tabs
```

### yarn

```bash
yarn add @spkf/react-tabs
```

### bun

```bash
bun add @spkf/react-tabs
```
